# 格式转换工具 (Format Converter)

一个功能强大的音频和视频格式转换工具，提供直观的图形用户界面，支持多种格式的转换。

## 主要功能

### 1. 视频格式转换
- 支持多种视频格式之间的相互转换
- 可自定义视频编码格式和码率
- 提供硬件加速选项

### 2. 音频格式转换
- 支持多种音频格式之间的相互转换
- 可自定义音频编码格式和码率

### 3. 批量处理
- 支持批量转换多个文件
- 可设置批量处理阈值
- 可调整最大线程数以优化性能

### 4. 界面特性
- 直观的图形用户界面
- 标签页式设置界面，便于导航
- 全界面使用微软雅黑字体，确保清晰显示
- 实时状态显示，了解转换进度

## 支持的格式

### 视频格式
- MP4
- AVI
- MKV
- MOV
- WMV
- FLV
- WebM

### 音频格式
- MP3
- WAV
- AAC
- FLAC
- OGG
- WMA

## 使用方法

1. 启动程序，在主界面选择输入格式和输出格式
2. 点击"选择输入文件"按钮，选择要转换的文件
3. 点击"选择输出路径"按钮，设置转换后的文件保存位置
4. 点击"转换"按钮开始转换过程
5. 转换完成后，程序会显示转换成功的提示

## 设置选项

通过"设置"按钮可以访问详细的配置选项：

### 视频设置
- 视频编码：选择视频编码格式（如libx264、H.265等）
- 视频码率：设置视频质量（如2000k、5000k等）

### 音频设置
- 音频编码：选择音频编码格式（如aac、mp3等）
- 音频码率：设置音频质量（如128k、192k等）

### 批量设置
- 批量处理阈值：设置一次批量处理的最大文件数
- 最大线程数：设置同时转换的线程数

### 高级设置
- 处理方式：选择使用CPU或GPU加速（如果支持）

## 安装说明

### 直接使用
1. 下载dist目录中的`format_converter_with_gui.exe`文件
2. 双击运行即可，无需安装

### 从源代码运行
1. 确保已安装Python 3.11或更高版本
2. 安装依赖：`pip install -r requirements.txt`
3. 运行程序：`python format_converter_with_gui.py`
